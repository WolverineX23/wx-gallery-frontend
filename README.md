# 项目初始化
- create-vue 脚手架
```sh
npm create vue@latest
```
> 构建选项：TS、Vue Router、Pinia、Prettier

## 命令
- vite
```sh
npm install
```
### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
